# Last

Find the last element of the given argument(s).

<b>Examples</b>
```
last(Arrays.asList(1, 2, 3, 4)); // =>  4
last("xyz");                     // => "z"
last(1, 2, 3, 4);                // =>  4
last(new int[]{1, 2, 3, 4});     // =>  4
```
In <b>javascript</b> and <b>CoffeeScript</b> a list will be an `array`, a `string` or the list of `arguments`.
